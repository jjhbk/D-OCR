# D-OCR

A decentralized OCR solution for transparent document compilation

## Cartesi Dapp

A decentralized application built using [cartesi](cartesi.io) & [tesseract](https://github.com/tesseract-ocr/tesseract)

### Building the dapp

```
cartesi build

cartesi run --verbose
```

## Building Frontend

```
cd frontend
yarn
yarn run dev

```
